# NGJA Gem & Jewellery Export Invoice System

Frontend-only premium export invoice workspace built with React, Vite, Tailwind CSS, React Router, Framer Motion, and Lucide icons.

## Highlights

- Role-based workspace selection (Admin / User)
- Dealer registration and verification status flow
- Premium dashboard layouts with glassmorphism cards
- Template selection and Template 1 invoice workspace
- Dynamic valuation table with auto totals
- Mock data, local state only, no authentication

## Run Locally

```bash
npm install
npm run dev
```

## Notes

- This project uses hardcoded data and mock logic only.
- All pages are prepared for future API integration.
